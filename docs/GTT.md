  两个问题：

---

## GTT 是什么

**GTT = Graphics Translation Table**，AMD GPU 的内存管理机制。

### 作用

| 内存区域 | 说明 |
|---------|------|
| **VRAM** | GPU 直接访问的物理内存（独显的 GDDR，或 iGPU 的 BIOS 预留区）|
| **GTT** | GPU 通过页表映射访问**系统内存**的窗口 |

```
iGPU 内存架构：
┌─────────────────┐
│   BIOS 预留区   │  ← "VRAM"（你的 4GB）
│  (GPU 优先访问)  │
├─────────────────┤
│   系统内存      │  ← "GTT" 可映射的区域（你的 ~14GB）
│  (CPU/GPU 共享)  │
└─────────────────┘
```

**GTT 让 iGPU 能使用远超 BIOS 预留区的系统内存**，但速度受 DDR5 带宽限制。

---

## APU 是什么

**APU = Accelerated Processing Unit**，AMD 的 CPU+GPU 融合芯片。

### 不是"低级 GPU"

| 特性 | 说明 |
|------|------|
| **设计目标** | 低功耗、集成、低成本 |
| **性能** | 远弱于同代独显，但强于纯 CPU 核显 |
| **定位** | 笔记本、迷你主机、入门台式机 |
| **你的 780M** | RDNA 3 架构，约 GTX 1050 Ti 水平 |

### 对比

| | APU (780M) | 入门级独显 (RX 6400) | 中端独显 (RTX 3060) |
|--|-----------|---------------------|-------------------|
| **显存** | 共享 DDR5 | 4GB GDDR6 | 12GB GDDR6 |
| **带宽** | ~50 GB/s | ~64 GB/s | ~360 GB/s |
| **算力** | ~8 TFLOPS | ~5 TFLOPS | ~25 TFLOPS |
| **功耗** | 15-30W | 50W | 170W |
| **价格** | 含在 CPU 中 | ~$150 | ~$350 |

**APU 不是"低级"**，是**能效比优先**的设计。780M 在 15W 下提供接近 50W 独显的性能。

---

## 为什么 APU 用 GTT

| 原因 | 说明 |
|------|------|
| **没有独立显存芯片** | 节省成本和主板空间 |
| **灵活分配** | 系统内存按需借给 GPU |
| **UMA 架构** | CPU/GPU 零拷贝共享数据 |

---

## 总结

| 术语 | 含义 |
|------|------|
| **GTT** | iGPU 访问系统内存的机制，决定实际可用"显存"上限 |
| **APU** | AMD CPU+GPU 融合芯片，不是低级，是低功耗高效能设计 |
| **你的 780M** | RDNA 3 APU，性能约 GTX 1050 Ti，适合推理入门 |

你的 780M 跑 VoxCPM 推理，属于 **"能跑但慢"** 的级别，不是不能用。
